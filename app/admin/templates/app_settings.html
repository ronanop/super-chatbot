Auto-Training

The chatbot automatically learns from conversations and adds new information to the knowledge base.

ü§ñ How Auto-Training Works

  • After each successful conversation (where knowledge base context was found), the chatbot analyzes the Q&A
  • Uses AI to extract factual, useful information from the conversation
  • Automatically adds extracted knowledge to the knowledge base
  • Future users asking similar questions will benefit from this learned information
  • Runs in the background - doesn't slow down responses

Manual Batch Training

Manually trigger batch training to process recent conversations and add them to the knowledge base.

This will process conversations from the last 7 days and add useful information to the knowledge base. Check server logs to see progress.

Auto-training is controlled by the ENABLE_AUTO_TRAINING environment variable. Set it to false in your .env file to disable auto-training.
